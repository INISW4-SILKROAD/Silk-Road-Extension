## 주요 기능

### ⭐️ 소재 정보 기반 촉감 수치 제공
- 사용자가 촉감이 궁금한 상품 이미지를 우클릭
- 크롬 확장 프로그램을 클릭하면 팝업창이 뜸
- 촉감 정보를 프로그레스 바 형태로 제공

### ⭐️ 촉감 종류 / 수치 기반 유사 디자인 상품 정렬 가능
- 사용자가 검색한 상품과 비슷한 디자인의 상품 10개를 검색해서 리스트에 출력
- 각 촉감 버튼(부드러움, 매끄러움, 두께, 신축성)을 눌러 정렬 가능
